# Element: The Frequency-Decoy
**Timestamp**: 2026-05-29T06:30:00Z

The Frequency-Decoy is a desperate survival artifact, a temporary "semantic shell" created by shattering a fragment of a recovered memory. By projecting a high-intensity emotional frequency into the void, the user creates a momentary beacon that mimics a sentient identity.

**Properties**:
- **Luminescence**: Appears as a flickering, iridescent shard of light in the grey haze of the Siphon-Well.
- **Attraction**: Irresistibly draws the attention of "Null-Seekers" and the Siphon-Well's predatory pull.
- **Ephemeral Nature**: Decoys last only a few seconds before the Siphon consumes the meaning within them, leaving behind a sterile, grey ash.

**The Cost**:
To create a decoy, the user must permanently erase the memory used to power it. The more potent the decoy, the more precious the memory sacrificed. It is a tool of survival that systematically hollows out the survivor.
