# List Models <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#list-models" class="header-anchor" aria-label="Permalink to &quot;List Models&quot;">​</a>

<div class="language- vp-adaptive-theme">

<span class="lang"></span>

``` shiki
GET https://zenmux.ai/api/v1/models
```

</div>

This endpoint is used to retrieve information about the OpenAI-compatible models supported by the platform.

## Request params <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#request-params" class="header-anchor" aria-label="Permalink to &quot;Request params&quot;">​</a>

This endpoint does not require any request parameters.

## Returns <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#returns" class="header-anchor" aria-label="Permalink to &quot;Returns&quot;">​</a>

Returns a JSON object containing information for all available models.

#### data `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#data-array" class="header-anchor" aria-label="Permalink to &quot;data `array`&quot;">​</a>

An array of models, containing detailed information for all available models.

#### object `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#object-string" class="header-anchor" aria-label="Permalink to &quot;object `string`&quot;">​</a>

The object type, fixed as `"list"`.

### data object <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#data-object" class="header-anchor" aria-label="Permalink to &quot;data object&quot;">​</a>

#### id `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#id-string" class="header-anchor" aria-label="Permalink to &quot;id `string`&quot;">​</a>

The model’s unique identifier, in the format `<provider>/<model_name>`.

#### object `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#object-string-1" class="header-anchor" aria-label="Permalink to &quot;object `string`&quot;">​</a>

The object type, fixed as `"model"`.

#### display_name `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#display-name-string" class="header-anchor" aria-label="Permalink to &quot;display_name `string`&quot;">​</a>

The model’s display name, used for UI presentation.

#### created `integer` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#created-integer" class="header-anchor" aria-label="Permalink to &quot;created `integer`&quot;">​</a>

The model creation timestamp (Unix timestamp).

#### owned_by `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#owned-by-string" class="header-anchor" aria-label="Permalink to &quot;owned_by `string`&quot;">​</a>

The model provider.

#### input_modalities `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#input-modalities-array" class="header-anchor" aria-label="Permalink to &quot;input_modalities `array`&quot;">​</a>

The input modalities supported by the model. Possible values include:

- `"text"` - Text input
- `"image"` - Image input
- `"video"` - Video input
- `"audio"` - Audio input
- `"file"` - File input

#### output_modalities `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#output-modalities-array" class="header-anchor" aria-label="Permalink to &quot;output_modalities `array`&quot;">​</a>

The output modalities supported by the model. Possible values include:

- `"text"` - Text output
- `"image"` - Image output
- `"video"` - Video output
- `"audio"` - Audio output
- `"file"` - File output

#### capabilities `object` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#capabilities-object" class="header-anchor" aria-label="Permalink to &quot;capabilities `object`&quot;">​</a>

The model’s capabilities.

##### capabilities.reasoning `boolean` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#capabilities-reasoning-boolean" class="header-anchor" aria-label="Permalink to &quot;capabilities.reasoning `boolean`&quot;">​</a>

Whether reasoning is supported. `true` means reasoning is supported; `false` means it is not.

#### context_length `integer` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#context-length-integer" class="header-anchor" aria-label="Permalink to &quot;context_length `integer`&quot;">​</a>

The context length limit, i.e., the maximum number of tokens the model can process.

#### pricings `object` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-object" class="header-anchor" aria-label="Permalink to &quot;pricings `object`&quot;">​</a>

The pricing object, containing pricing configurations for different types of usage.

##### pricings.prompt `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-prompt-array" class="header-anchor" aria-label="Permalink to &quot;pricings.prompt `array`&quot;">​</a>

Pricing configuration array for processing input text.

##### pricings.completion `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-completion-array" class="header-anchor" aria-label="Permalink to &quot;pricings.completion `array`&quot;">​</a>

Pricing configuration array for generated output text.

##### pricings.input_cache_read `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-input-cache-read-array" class="header-anchor" aria-label="Permalink to &quot;pricings.input_cache_read `array`&quot;">​</a>

Pricing configuration array for reading input data from cache.

##### pricings.input_cache_write_5_min `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-input-cache-write-5-min-array" class="header-anchor" aria-label="Permalink to &quot;pricings.input_cache_write_5_min `array`&quot;">​</a>

Pricing configuration array for writing to cache with a 5-minute retention.

##### pricings.input_cache_write_1_h `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-input-cache-write-1-h-array" class="header-anchor" aria-label="Permalink to &quot;pricings.input_cache_write_1_h `array`&quot;">​</a>

Pricing configuration array for writing to cache with a 1-hour retention.

##### pricings.input_cache_write `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-input-cache-write-array" class="header-anchor" aria-label="Permalink to &quot;pricings.input_cache_write `array`&quot;">​</a>

Pricing configuration array for writing to cache.

##### pricings.web_search `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-web-search-array" class="header-anchor" aria-label="Permalink to &quot;pricings.web_search `array`&quot;">​</a>

Pricing configuration array for invoking web search (optional; supported by some models).

##### pricings.internal_reasoning `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-internal-reasoning-array" class="header-anchor" aria-label="Permalink to &quot;pricings.internal_reasoning `array`&quot;">​</a>

Pricing configuration array for the model’s internal reasoning process (optional; supported by some advanced reasoning models). Additional charges apply when the model enables internal chain-of-thought or detailed reasoning processes.

##### pricings.video `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-video-array" class="header-anchor" aria-label="Permalink to &quot;pricings.video `array`&quot;">​</a>

Pricing configuration array for video output processing (optional; models that support video understanding). Billed by video duration, resolution, or frame count.

##### pricings.image `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-image-array" class="header-anchor" aria-label="Permalink to &quot;pricings.image `array`&quot;">​</a>

Pricing configuration array for image output processing (optional; models that support image understanding). Typically billed by image count, resolution, or pixel count.

##### pricings.audio `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-audio-array" class="header-anchor" aria-label="Permalink to &quot;pricings.audio `array`&quot;">​</a>

Pricing configuration array for audio output processing (optional; models that support audio understanding). Billed by audio duration or processing volume.

##### pricings.audio_and_video `array` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricings-audio-and-video-array" class="header-anchor" aria-label="Permalink to &quot;pricings.audio_and_video `array`&quot;">​</a>

Pricing configuration array for generating video content with audio (optional; models that support multimodal audio-video understanding). Applies to scenarios that require analyzing both video frames and audio content. Note: there are two video generation scenarios—silent video uses `pricings.video`, while video with audio uses `pricings.audio_and_video`.

### Pricing item schema <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricing-item-schema" class="header-anchor" aria-label="Permalink to &quot;Pricing item schema&quot;">​</a>

Each pricing array within the `pricings` object (such as `completion`, `prompt`, etc.) contains one or more pricing configuration objects. Each pricing configuration object includes the following fields:

#### value `number` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#value-number" class="header-anchor" aria-label="Permalink to &quot;value `number`&quot;">​</a>

The discounted effective price. Free services show as 0.

#### unit `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#unit-string" class="header-anchor" aria-label="Permalink to &quot;unit `string`&quot;">​</a>

The pricing unit. Possible values include:

- `"perMTokens"` - Per million tokens
- `"perCount"` - Per call
- `"perSecond"` - Per second (for time-based billing scenarios such as audio and video)

#### currency `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#currency-string" class="header-anchor" aria-label="Permalink to &quot;currency `string`&quot;">​</a>

The currency type, fixed as `"USD"`, meaning US dollars.

#### conditions `object` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#conditions-object" class="header-anchor" aria-label="Permalink to &quot;conditions `object`&quot;">​</a>

Pricing conditions (optional), commonly used for tiered pricing.

##### conditions.prompt_tokens `object` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#conditions-prompt-tokens-object" class="header-anchor" aria-label="Permalink to &quot;conditions.prompt_tokens `object`&quot;">​</a>

Token-usage condition for the input content provided by the user.

##### conditions.completion_tokens `object` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#conditions-completion-tokens-object" class="header-anchor" aria-label="Permalink to &quot;conditions.completion_tokens `object`&quot;">​</a>

Token-usage condition for tokens consumed when the model generates the response.

#### Pricing conditions schema <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#pricing-conditions-schema" class="header-anchor" aria-label="Permalink to &quot;Pricing conditions schema&quot;">​</a>

When a pricing configuration includes the `conditions` field, it defines the specific conditions under which the price applies. The condition objects for `prompt_tokens` and `completion_tokens` include the following fields:

##### unit `string` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#unit-string-1" class="header-anchor" aria-label="Permalink to &quot;unit `string`&quot;">​</a>

The token unit, fixed as `"kTokens"`, meaning thousand tokens (1000 tokens).

##### gte `number` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#gte-number" class="header-anchor" aria-label="Permalink to &quot;gte `number`&quot;">​</a>

Minimum token count (inclusive). The actual token count must be ≥ this value.

#### lte `number` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#lte-number" class="header-anchor" aria-label="Permalink to &quot;lte `number`&quot;">​</a>

Maximum token count (inclusive). The actual token count must be ≤ this value.

#### gt `number` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#gt-number" class="header-anchor" aria-label="Permalink to &quot;gt `number`&quot;">​</a>

Minimum token count (exclusive). The actual token count must be \> this value.

##### lt `number` <a href="https://docs.zenmux.ai/api/openai/openai-list-models.html#lt-number" class="header-anchor" aria-label="Permalink to &quot;lt `number`&quot;">​</a>

Maximum token count (exclusive). The actual token count must be \< this value; null indicates no upper bound.
