# <a href="https://cljdoc.org/d/lilactown/helix/0.2.2/doc/readme#helix" id="helix" class="md-anchor">Helix</a>

ClojureScript optimized for modern React development.

``` clojure
(ns my-app.core
  (:require [helix.core :refer [defnc $]]
            [helix.hooks :as hooks]
            [helix.dom :as d]
            ["react-dom/client" :as rdom]))

;; define components using the `defnc` macro
(defnc greeting
  "A component which greets a user."
  [{:keys [name]}]
  ;; use helix.dom to create DOM elements
  (d/div "Hello, " (d/strong name) "!"))

(defnc app []
  (let [[state set-state] (hooks/use-state {:name "Helix User"})]
    (d/div
     (d/h1 "Welcome!")
      ;; create elements out of components
      ($ greeting {:name (:name state)})
      (d/input {:value (:name state)
                :on-change #(set-state assoc :name (.. % -target -value))}))))

;; start your app with your favorite React renderer
(defonce REDACTED_SECRET (rdom/createRoot (js/document.getElementById "app")))
(.render REDACTED_SECRET ($ app))
```

## <a href="https://cljdoc.org/d/lilactown/helix/0.2.2/doc/readme#installation" id="installation" class="md-anchor">Installation</a>

<a href="https://clojars.org/lilactown/helix" rel="nofollow"><img src="https://img.shields.io/clojars/v/lilactown/helix.svg" alt="Clojars Project" /></a>

Install the latest version from clojars in your project.

Install JS dependencies:

``` clojure
npm init # initialize NPM project if necessary
npm i react react-refresh # install react, and react-refresh for hot reloading support (see docs)
npm i react-dom # install renderer. alternatives could be react-native, react-three-fiber, etc.
```

### <a href="https://cljdoc.org/d/lilactown/helix/0.2.2/doc/readme#shadow-cljs-and-react-native" id="shadow-cljs-and-react-native" class="md-anchor">shadow-cljs and react-native</a>

See [React Native](https://cljdoc.org/d/lilactown/helix/0.2.2/doc/example-projects-).

### <a href="https://cljdoc.org/d/lilactown/helix/0.2.2/doc/readme#lein-cljsbuild--figwheel-main--raw-cljs" id="lein-cljsbuild--figwheel-main--raw-cljs" class="md-anchor">lein-cljsbuild / figwheel-main / raw CLJS</a>

Use <a href="https://github.com/cljsjs/packages/tree/master/react" rel="nofollow">CLJSJS</a> or package react yourself using webpack, ensuring it is provided as the name `"react"`.
