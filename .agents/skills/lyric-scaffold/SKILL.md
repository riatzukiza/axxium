# Skill: lyric-scaffold

## Goal
Produce a syllable-aware lyric skeleton that fits over a generated beat's bar count and tempo.

## Use This Skill When
- Asked for lyrics, rap bars, a song concept, or vocal melody hooks.
- Attaching words to a beat already generated by beat-compose.
- Building a Discord bot that responds with words + music.

## Do Not Use This Skill When
- Pure instrumental is requested.
- Full lyric writing (this produces scaffold only — structure, not finished lines).

## Inputs
- Beat parameters: BPM, bar count, feel (from mood-to-music or beat-compose).
- Genre: rap | pop | rnb | ambient | spoken-word.
- Theme or subject matter.

## Steps
1. Calculate syllables-per-bar: `(BPM / 60) * beats_per_bar * syllables_per_beat`.
   - Rap: ~4–6 syllables/beat. Pop: ~2–3.
2. Emit structure: `[intro, verse1, prechorus, chorus, verse2, chorus, bridge, chorus, outro]`.
3. For each section: label, bar count, rhyme scheme (AABB / ABAB / ABCB), syllable budget.
4. Add 2–3 placeholder lines per section showing rhythm pattern.

## Output
```
[CHORUS] 4 bars | AABB | ~16 syl/line
_DA-da-da-DA-da-da-DA-da_ (x2)
Line A: [16 syllables, ends with rhyme-set-1]
Line B: [16 syllables, ends with rhyme-set-1]
```
